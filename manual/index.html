    Remote-lib

    Remote-lib is a library that can run remotely on other node.js servers or even on remote clients on the browser. It's only need a Duplex stream such as TCP soocket, WebSocket or even WebRTC DataChannel.

    Install

    npm install remote-lib

    API Reference

    Remote-lib is build with many small sub-packages, each package implement a small part of this library.

    • remote-lib - A high level API for creating remote libraries.
    • remote-context - The core of remote-lib, creating and serving remote context.
    • remote-environment - A shared environment context with native objects such as Object, Number, String, etc...
    • remote-instance - A stream transformer that can parse and construct instance remotely.
    • remote-protocol - The core of remote-lib protocol.
    • reference-context - Virtual context implementation on vanilla Javascript.
